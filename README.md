下記の技術を学習するために、作成しました。

# 使用技術
| 技術名 | 役割 |
| ---- | ---- |
| Ruby on Rails | APIサーバー |
| 認証 | JWT(Rails gem devise_token_auth を使用) |
| Rails test | RSpec |
| Nuxt.js | フロントエンド |
| MySQL | DB |
| GraphQL, Appolo | query |
| localstack | 画像保存用 |
| Redis | NoSQL |

| localstack | 4566 & 8080 |
